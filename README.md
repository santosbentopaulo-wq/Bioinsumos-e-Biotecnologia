# 🌱 Bioinsumos e Biotecnologia — Agrinho 2026

Site educativo simples e responsivo sobre **Bioinsumos e Biotecnologia**, desenvolvido para o programa **Agrinho 2026**. Apresenta conteúdo informativo em linguagem acessível para estudantes e um quiz interativo para testar o aprendizado.

## 📁 Estrutura dos arquivos

```
├── index.html   → Estrutura da página (conteúdo e quiz)
├── style.css    → Estilos visuais e responsividade
└── script.js    → Lógica do quiz interativo
```

## 🚀 Como usar

1. Baixe os três arquivos (`index.html`, `style.css`, `script.js`) e mantenha-os na **mesma pasta**.
2. Abra o arquivo `index.html` em qualquer navegador (Chrome, Firefox, Edge etc.).
3. Pronto! O site funciona totalmente offline, sem necessidade de internet ou instalação.

## 🧩 Funcionalidades

- **Cabeçalho** com título e descrição do tema.
- **Seção informativa** explicando, de forma simples, o que são bioinsumos e biotecnologia e por que são importantes.
- **Quiz interativo** com 5 perguntas de múltipla escolha:
  - Feedback imediato (certo ✅ ou errado ❌) ao responder.
  - Destaque visual da resposta correta.
  - Pontuação final exibida ao término.
  - Botão para refazer o quiz.
- **Rodapé** com créditos ao Agrinho 2026.
- **Design responsivo**, adaptado para desktop e celular.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (puro, sem frameworks)
- JavaScript (puro, sem bibliotecas externas)

## ✏️ Personalização

- **Perguntas do quiz**: edite o array `perguntas` no início do arquivo `script.js`.
- **Cores e fontes**: ajuste as variáveis de cor e propriedades no arquivo `style.css`.
- **Textos informativos**: edite diretamente as seções `<section class="info">` no `index.html`.
